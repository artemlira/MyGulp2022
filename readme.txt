1. Работа с картинками: все необходимые картинки нужно положить в папку "img" в корне "src", путь к картинкам нужно прописывать через @, напр.: <img src="@img/logo.png" alt="">. Для того что бы работала авто замена картинок на webp в бекграунде, то свойство бекграунд нужно прописывать в файле style.scss в формате: "background:  url('@img/logo.png') no-repeat";

2. Для создания других страниц, таких как "О нас" или "Контакты" необходимо в корне папки "src" создать соответствующие файлы HTML, с таким же head header и footer как и в index.html, только в head нужно поменять заголовок

3. Для локального подключения шрифтов, необходимые шрифты нужно положить в папку "Font" в формате ttf или otf, потом перезапустить Gulp. После этого в папке Build появиться папка с шрифтами в формате woff и woff2, а в папке scss появиться файл fonts.scss который нужно будет подключить в файле style.scss

4. SVG Sprite: для создания спрайта необходимые иконки нужно загрузить в папку "svgicons" в корне "src", после этого запустить команду "npm run svgSprive". Готовый спрайт появиться в дериктории dist > img > icons > icons.svg
